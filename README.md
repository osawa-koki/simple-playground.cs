# simple-playground.cs

C#の学習用プロジェクトです。

各章ごとにプロジェクトを分けているため、機能ごとに調査実行をすることができます。  
※ ここでいう、プロジェクトとはC#プログラムの実行単位のことで、Gitのソースコード管理単位とは異なります。

| .NET | 6.0 |
